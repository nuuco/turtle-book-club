- 어휘력이 바닥을 기는 요즘, 사놓고 안 본 책들을 해치워버리기로 결심했다.
- 목표는 책장의 대략 30권이 넘는 안 읽은 책들을 다 읽어버리기

## 활동 규칙
1. 매일 책을 한 페이지라도 읽는다.
2. 인상깊은 문장을 쓴다.
3. 내 생각을 세 줄 이상 쓴다.
